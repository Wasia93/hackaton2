## Recording Instructions

### Option 1: Screen Recording (Manual)

**Tools**:
- **Windows**: Xbox Game Bar (Win + G) or OBS Studio
- **Mac**: QuickTime Player or Screen Studio
- **Cross-platform**: OBS Studio (free)

**Steps**:
1. Open terminal and navigate to project
2. Start recording
3. Follow script above
4. Practice 2-3 times first to get timing right
5. Edit out mistakes if needed
6. Export as MP4

### Option 2: NotebookLM Audio + Screen Recording

**Steps**:
1. Create NotebookLM notebook with this script
2. Generate audio podcast (auto-narration)
3. Record screen while playing audio
4. Sync actions with audio narration
5. Combine in video editor

### Option 3: AI Video Generation

**Tools**:
- **Loom** - Record screen + webcam
- **Descript** - AI-powered video editing with script
- **Synthesia** - AI avatar (if you don't want to narrate)

## Recording Tips

### Visual
- Use **fullscreen terminal** or **zoom in** so text is readable
- Clear, simple terminal prompt
- Highlight important text if possible
- Smooth, deliberate typing (not too fast)

### Audio
- Speak clearly and at moderate pace
- Enthusiasm! Show excitement about the project
- No background noise
- Use a good microphone (built-in is okay)

### Editing
- Cut out long pauses
- Speed up typing if needed (1.5x)
- Add text overlays for key points (optional)
- Fade in/out at start/end
